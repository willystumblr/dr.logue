# 문제3. 무음 구간 찾기

1. environment
	1) Create conda env
		$ conda create -n q3_env python==3.9
		$ conda activate q3_env

	2) Install required packages using requirements.txt
		$ pip3 install -r requirements.txt

2. how to start
	In the directory where Q3.py exists, enter the following command:
   	$ python3 Q3.py {path-to-wavlist.txt} {path-to-output-file}
	e.g., python3 Q3.py ./pcmlist.txt ../../output/Q3.json
